# QA Verify

Use this skill before declaring a feature done.

Read:
- AGENTS.md
- docs/PRODUCT.md
- docs/ARCHITECTURE.md
- latest docs/ITERATION_LOG.md entry

Verify:
- lint
- typecheck
- tests if present
- main affected flow
- obvious data consistency issues

Return:
- commands run
- pass/fail
- smoke-tested flows
- issues found
- fixes applied
- residual risks
